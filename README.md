# linview UI组件库

### 快速开始

1. #### 安装组件库

 ```shell
   npm i linview
 ```

   

2. #### 引用组件库

```js
//引入全部
import 'linview/dist/css/index.css';
import lview from 'linview';
Vue.use(lview)

//按需引入
import 'linview/dist/css/card.css'
import Card from 'linview/dist/card.umd';
Vue.use(Card)
```

官方文档：https://lsfcxz.github.io/linview
